# Bootstrap4

## Description

Bootstrap4 is a starter template using Bootstrap 4 — a simple foundation for building responsive, modern static websites. It helps you quickly launch landing pages, portfolios, or small projects without heavy setup.

## Why Use Bootstrap4

- Provides a ready-to-use HTML structure linked with Bootstrap 4 CSS/JS  
- Lets you build responsive pages right away (mobile-friendly by default)  
- Easy to customize: change markup, styles, or add your own components  
- Great for prototyping, simple websites, or learning Bootstrap  

## Repository Structure

Bootstrap4/
├── index.html ← Main HTML page using Bootstrap 4
├── css/ ← Optional custom styles (if any)
├── js/ ← Optional JS or plugin scripts (if any)
├── imgs/ ← Images / assets (optional)
└── README.md ← This README file
